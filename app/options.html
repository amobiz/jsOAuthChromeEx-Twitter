jsOAuthChromeEx.js Example - Twitter

jsOAuth is a JavaScript implimentation of the OAuth protocol. Currently supports version 1.0 (RFC5849) of the specification.

The jsOAuth plugin for Google Chrome Extension is an extension that enables Google Chrome Extension support. Please checkout README for full description.

You'll need to create an account on Twitter.

And then "Create New App".

Make sure the Callback URL field has a valid value (for chrome extension, this url is not invoked).

Fill in the consumerKey and consumerSecret in the form below. (Normally you will put these keys directly somewhere in your code.)
